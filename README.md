# BCA Vault - A Hibro Labs Venture

BCA Vault is a secure document management web application designed to allow users to upload, manage, and share PDFs with robust access control and monitoring features.

## Live Demo

Experience the live application here: [BCA Vault](https://bca-for-friends.vercel.app/)

## Overview

BCA Vault provides a centralized platform for storing important PDFs securely. The app features user authentication, activity tracking, an admin panel for user and document management, and detailed analytics to monitor usage.

## Features

- Secure login system with email and password authentication
- User profile dashboard displaying member info and recent activity
- Upload and manage PDFs with ease
- Admin panel for user management and activity log monitoring
- Detailed analytics dashboard showing total users, PDFs, and views
- Responsive design optimized for desktop and mobile
- Clean, intuitive interface for smooth user experience
